### 📄 Hybrid BERT-T5 Mixture of Experts (MoE) Model

This project introduces a **Hybrid BERT-T5 Mixture of Experts (MoE)** model designed for advanced educational NLP applications.

By combining **BERT’s deep contextual understanding** with **T5’s strong generative capabilities**, the model produces outputs that are both **semantically accurate** and **fluently constructed**.

#### 🚀 Key Features

* **Supervised MoE Router**
  Dynamically identifies the task type (e.g., summarization, question generation, question answering) and routes the input to the most suitable expert decoder.

* **Word-Level Representation Alignment**
  Aligns BERT and T5 representations at the word level to preserve meaning and contextual integrity.

* **Efficient Hybrid Architecture**
  The targeted fusion and routing strategy improves both computational efficiency and model performance.

#### 📊 Performance

* Evaluated on benchmark datasets
* Outperforms traditional transformer baselines
* Achieves better:

  * Accuracy
  * Coherence
  * Task routing effectiveness

#### 🎯 Conclusion

The proposed system demonstrates the potential of hybrid architectures in advancing **educational Natural Language Processing (NLP)**
